# AaveKit React Examples

This project contains examples using AaveKit React.


[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/aave/aave-v4-sdk/tree/main/examples/react-stories)


## Setup

1. Copy `.env.example` to `.env` and configure:

```bash
cp .env.example .env
```

2. Install dependencies:

```bash
pnpm install
```

3. Run the development server:

```bash
pnpm dev
```
